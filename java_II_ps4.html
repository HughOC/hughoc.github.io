Java II
Homework 4

Everyone remembers those cheap plastic tile games you got free with things as a child. They typically had 4 x 4 rectangular tiles organized in a grid. You had to move the pieces up, down, left or right until you revealed the unjumbled image.

That was what we had to make in Java II’s last problem sheet. We had to create a 4 x 4 grid of squares numbered 1 to 15 (where the 16th square had no tile so that the other tiles could move around).

One interesting note to make about this game is that when you open it up the tiles are arranged from 1 to 15 is the correct order. In other words, the game is complete when you open it. In order to make the game playable we had to make a shuffle button that jumbled the tiles up for the user to solve.
